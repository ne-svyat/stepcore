# StepCore — обязательная настройка на MIUI/HyperOS

Без этих настроек MIUI душит фоновую работу. Проверять после
обновлений прошивки — MIUI любит сбрасывать.

1. ЗАМОК В НЕДАВНИХ (главное).
   Открыть список запущенных приложений → нажать и ДЕРЖАТЬ карточку
   StepCore → нажать иконку замка 🔒. На карточке должен появиться
   замочек. Без него MIUI выгружает приложение при блокировке экрана.

2. Настройки → Приложения → StepCore:
   - Автозапуск: ВКЛ
   - Экономия энергии: Без ограничений

3. Приложение «Безопасность» → Батарея → шестерёнка:
   - «Очищать память при блокировке экрана» — ВЫКЛ
     (или добавить StepCore в исключения).

4. При первом нажатии «Старт» разрешить все запрошенные права.
