# Корпус текстов Survival Mode — SM-2 (Северная тайга, погода).
#
# Формат:   ключ [условия] | текст
# Условия необязательны. Строка без условий годится для любого дня.
# Строка с условиями выпадает ТОЛЬКО в день, где выполнены ВСЕ условия.
#
# Что можно писать в условиях:
#   t       — температура дня          (t<=-5, t>=20)
#   dt      — насколько холоднее вчера (dt<=-10)
#   season  — winter / spring / summer / autumn
#   cloud   — 0 ясно · 1 переменная · 2 пасмурно
#   wind    — 0 тихо · 1 ветрено · 2 сильный · 3 буря
#   precip  — 0 нет · 1 дождь · 2 ливень · 3 мокрый снег · 4 снег
#   fog     — 1 туман · 0 нет
#   snowseen— 1 снег в эту экспедицию уже выпадал
#   snowcm  — глубина снежного покрова на земле, см (0 = земля голая)
#   ice     — 0 река открыта · 1 шуга/забереги · 2 ледостав
#   winter  — 1 зима фактически встала (снег лежит, мороз держит)
#
# Правило: если строка упоминает снег, мороз, иней, наст — она ОБЯЗАНА
# нести условие. Иначе она однажды выпадет в июльскую жару.
#
# Тон: сдержанный полевой дневник. Природа безразлична. Без восклицаний.
